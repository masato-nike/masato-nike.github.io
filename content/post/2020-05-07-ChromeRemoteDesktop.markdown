新型コロナウイルスの影響で、自宅から会社や友人宅のパソコンを操作したいなという機会が増えたので、Google Chromeの拡張機能であるリモートデスクトップのインストール方法を覚書代わりに書いておきます。<br />
<!--more-->
(1) 以下のURLにGoogle Chromeでアクセスしてください。<br />
<br />
[Chromeウェブストア](https://chrome.google.com/webstore/category/extensions?hl=ja "Chromeウェブストア")<br />
<br />
(2) 検索窓にて"remote desktop"と入力してください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_001.JPG "Google Chrome Remote Desktop Web") <br /><br />
(3) Chrome Remote Desktopの[Chromeに追加]をクリックしてください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_002.JPG "Google Chrome Remote Desktop Web") <br /><br />
(4) [拡張機能を追加]をクリックしてください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_003.JPG "Google Chrome Remote Desktop Web") <br /><br />
(5) 下の画像の赤丸のアイコンをクリックしてください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_004.JPG "Google Chrome Remote Desktop Web") <br /><br />
(6) ダウンロードボタン（↓矢印）をクリックしてください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_005.JPG "Google Chrome Remote Desktop Web") <br /><br />
(7) [同意してインストール]をクリックしてください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_006.JPG "Google Chrome Remote Desktop Web") <br /><br />
(8) [はい]をクリックしてください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_007.JPG "Google Chrome Remote Desktop Web") <br /><br />
(9) [はい]をクリックしてください。<br />
![Google Chrome Remote Desktop Web](/images/blog/ChromeRemoteDesktop/ChromeRemoteDesktop_008.JPG "Google Chrome Remote Desktop Web") <br /><br />
以上でChrome Remote Desktopのインストールは完了です。
